# MousePoint

Windows 강의/프레젠테이션용 화면 오버레이 주석 도구.

마우스를 따라다니는 레이저 포인터, 형광펜, 네모박스로 화면 위에 직접 그릴 수 있습니다.
PPT 전체화면, Zoom 화면공유, 멀티모니터 환경에서 모든 앱 위에 동작합니다.

## 다운로드

[Releases](https://github.com/hyunsungko/mousepoint/releases) 페이지에서 **MousePoint.exe**를 다운로드하세요.
설치가 필요 없는 단일 실행파일입니다. 다운로드 후 바로 실행하면 됩니다.

### 시스템 요구사항

- Windows 10 또는 Windows 11 (64비트)
- .NET 8 런타임이 포함되어 있어 별도 설치 불필요

### 처음 실행 시 보안 경고

코드 서명이 없는 오픈소스 프로그램이라 처음 실행할 때 "Windows가 PC를 보호했습니다" 경고가 나타날 수 있습니다.

1. **추가 정보**를 클릭합니다
2. **실행**을 클릭합니다

한 번만 하면 되고, 이후에는 경고 없이 실행됩니다.

## 사용법

### 시작/종료

| 입력 | 동작 |
|------|------|
| **F9** | 오버레이 켜기/끄기 (끄면 마우스가 정상으로 돌아옴) |
| **ESC** | 오버레이 끄기 |
| **Ctrl+Shift+Q** | 프로그램 완전 종료 |

### 도구 전환

| 입력 | 동작 |
|------|------|
| **마우스 사이드 버튼 1** | 도구 순환: 레이저 → 형광펜 → 네모박스 → 끄기 |
| **마우스 사이드 버튼 2** | 현재 도구의 색상 변경 (4색 순환) |
| **스크롤 휠** | 형광펜 굵기 / 네모박스 테두리 굵기 조절 |

사이드 버튼이 없는 마우스는 키보드로 대체할 수 있습니다:

| 입력 | 동작 |
|------|------|
| Ctrl+Shift+1 | 레이저 포인터 |
| Ctrl+Shift+2 | 형광펜 |
| Ctrl+Shift+3 | 끄기 (비활성) |
| Ctrl+Shift+4 | 네모박스 |

### 도구 설명

**레이저 포인터** — 마우스를 따라 발광하는 원이 표시됩니다. 꼬리가 자연스럽게 사라지는 그라데이션 효과. 4가지 색상(빨강, 초록, 파랑, 노랑).

**형광펜** — 좌클릭+드래그로 화면 위에 반투명 선을 그립니다. 3초 후 자동으로 사라집니다. 4가지 색상, 3단계 굵기(스크롤 휠로 조절).

**네모박스** — 좌클릭+드래그로 반투명 형광색 사각형을 그립니다. 강조하고 싶은 영역을 감쌀 때 유용합니다. 1.5초 후 자동으로 사라집니다. 4가지 색상, 3단계 테두리 굵기.

### 동작 방식

- **F9를 누르면** 레이저 포인터가 켜지고, 화면 위에 오버레이가 표시됩니다
- **오버레이가 켜져 있는 동안** 마우스 클릭은 아래의 앱으로 전달되지 않습니다
- **F9 또는 ESC를 누르면** 오버레이가 사라지고 마우스가 정상으로 돌아옵니다
- 그린 것들은 시간이 지나면 자동으로 사라집니다

## v0.2에서 변경된 점 (2026-04-01)

### 새 기능
- **네모박스 도구** — 드래그로 반투명 사각형 그리기 (둥근 모서리, 자동 fade-out)
- **레이저 포인터 4색** — 빨강/초록/파랑/노랑 + 발광 효과 + 그라데이션 꼬리
- **형광펜 굵기 조절** — 스크롤 휠로 3단계 변경
- **네모박스 테두리 굵기 조절** — 스크롤 휠로 3단계 변경
- **종료 단축키** — Ctrl+Shift+Q로 프로그램 종료, ESC로 오버레이 끄기
- **온보딩 가이드** — 처음 실행 시 조작법 안내 (한국어/영어)

### 성능 개선
- **DWM 하드웨어 가속 투명도** — 형광펜을 빠르게 연속으로 그려도 버벅임 없음
- **Douglas-Peucker 포인트 간소화** — 완성된 스트로크의 렌더링 부하 감소
- **드래그 중 fade-out 일시정지** — 그리는 동안 CPU 경합 방지

### 변경사항
- F9 동작 변경: 이전에는 클릭 통과 토글 → 이제는 오버레이 자체를 숨김/표시
- 레이저 포인터 중앙 투명도 50%로 부드러운 느낌
- 레이저 꼬리에 이차 곡선 그라데이션 + 굵기 테이퍼링 적용

## 알려진 제한사항

- **오버레이 활성 시 마우스 클릭 불가** — 레이저/형광펜/네모박스 모드에서는 아래의 앱을 클릭할 수 없습니다. F9 또는 ESC로 끄면 정상으로 돌아옵니다.
- **전체화면 DirectX 앱 위 표시 불가** — 게임 등 DirectX 전체화면 앱 위에서는 오버레이가 보이지 않을 수 있습니다. 창 모드에서는 정상 동작합니다.
- **SmartScreen 경고** — 코드 서명이 없어 처음 실행 시 보안 경고가 나타납니다. "추가 정보 → 실행"으로 해결됩니다.
- **관리자 권한 앱** — 관리자 권한으로 실행된 앱 위에서는 오버레이가 가려질 수 있습니다. MousePoint를 관리자 권한으로 실행하면 해결됩니다.
